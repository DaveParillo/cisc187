Detailed Description

Transform command line arguments passed to take-away into options the Controller can use.

The only justification for this class is to simplify passing initial values from main into the Controller class. And it's a fairly minor simplification...

size_t Options::_coins = _default_coins
private

The coins used in the current game.

If a non-random game is played, then this is the number of coins in every pile of the game. If a random game is played, then this is the maximum number of coins in each pile. Each pile will contain from 1 to N coins.
